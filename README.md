# 🏡 PropertyNest

PropertyNest is a modern full-stack real estate platform that allows users to discover, buy, sell, and manage properties across Bangladesh. The application provides a seamless experience for customers, property owners, and administrators with secure authentication, responsive UI, and powerful property management features.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/Monoaratasnim/SCIC13-A03-Client.git
```

```bash
cd propertynest-client
```

Install Dependencies

```bash
npm install
```

Run Development Server

```bash
npm run dev
```

---

# ⚙️ Environment Variables

## Frontend (.env.local)

```env
NEXT_PUBLIC_API_URL=http://localhost:5000/api
```

---

## Backend (.env)

```env
PORT=5000

MONGODB_URI=

JWT_SECRET=

CLIENT_URL=http://localhost:3000

GOOGLE_CLIENT_ID=

GOOGLE_CLIENT_SECRET=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
